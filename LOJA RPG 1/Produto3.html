DESCRIÇÃO GERAL

Nosso conjunto de dados para RPG contém os sete principais e mais tradicionais dados do mercado. Possui também uma das características mais importantes para se manter a aleatoriedade do resultado, que é estar bem equilibrado, além de possuir números legíveis para facilitar a leitura e um marcador para diferenciar o número 6 do número 9. Os dados são de alta qualidade, feitos de poliresina, considerado o melhor material para dados do mercado, já que seu diferencial é a durabilidade e a resistência.

Os principais dados utilizados são os D4, D6, D8, D10 padrão (0-9), D10 dezena (00-90), D12 e D20. O número depois do d representa a quantidade de faces que o dado tem.

Você pode ter notado que possui dois dados de dez lados, isso porque um deles é numerado com dezenas de 00 a 90 e é usado de uma maneira muito interessante. Realizando uma rolagem c\om os 2D10 (dois dados de 10 faces, um padrão e um de dezena) você consegue obter um resultado de porcentagem de 01% a 100%, conseguindo assim jogar diversos tipos de jogos (sistemas) de RPG.

O D20 é o principal dado de D&D e se tornou o símbolo do RPG.
